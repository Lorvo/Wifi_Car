The communication Is done using TCP protocol

I managed to drive it from a separate network using ngrok tunneling services

The servo is powered from the 5V pin on the board which caused power issues but adding a few capacitors helped and I don't get any more reboots while steering frantically

The steering lines on the client feed are not accurate and I lack the motivation to calibrate them (I added some crude callibration options)
